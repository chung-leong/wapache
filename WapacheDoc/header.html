HTTP Header Extensions

X-Command-After
X-Command-Before
X-Line-Handler
X-Command-After Header
Description: Carries out a command after the body of the response has been processed
Syntax: X-Command-After-#: command-type window-name args ...
Context: HTTP response

X-Command-Before Header
Description: Carries out a command before the body of the response has been processed
Syntax: X-Command-Before-#: command-type window-name args ...
Context: HTTP response

X-Line-Handler Header
Description: Redirects text in the response body to a scripting function
Syntax: X-Line-Handler: window-name function-name
Context: HTTP response
