DefaultEnv
Description: Default value of an Apache environment variable
Syntax: DefaultEnv variable-name value
Context: server config, virtual host

DocumentRoot
Description: Directory that forms the main document tree visible to Wapache
Syntax: DocumentRoot directory-path
Default: DocumentRoot <config-directory-path>/htdocs
Context: server config, virtual host

PersistentEnv
Description: Mark an environment variable as being persistent
Syntax: PersistentEnv variable-name
Context: server config, virtual host

ServerRoot
Description: Base directory for the server installation
Syntax: ServerRoot directory-path
Default: ServerRoot <executable-directory-path>
Context: server config

SessionEnv
Description: Indicates that an Apache environment variable should be preserved until the program exits
Syntax: SessionEnv variable-name
Context: server config, virtual host

<VirtualHost>
Description: Contains directives that apply only to a specific hostname
Syntax: <VirtualHost addr [addr] ...> ... </VirtualHost>
Context: server config

ActiveX
Description: Determines whether the web browser can load ActiveX controls
Syntax: ActiveX On|Off
Default: ActiveX On
Context: server config

DefaultIconPath
Description: Specifies the path to the default icon file
Syntax: DefaultIconPath ico-file-path
Default: DefaultIconPath (none)
Context: server config

ExternalObject
Description: Determines whether the external object is available for scripting in the web browser
Syntax: ExternalObject On|Off
Default: ExternalObject Off
Context: server config

InitialUrl
Description: Adds to the list of URLs that the program navigates to when it starts
Syntax: InitialUrl opener url [target]
Context: server config

Java
Description: Determines whether the web browser can run Java applets
Syntax: Java On|Off
Default: Java On
Context: server config

Scripting
Description: Determines whether the web browser can run Javascript and VBScript
Syntax: Scripting On|Off
Default: Scripting On
Context: server config

3DBorder
Description: Determines whether a 3D border appears around a browser frame
Syntax: 3DBorder On|Off
Default: 3DBroder On
Context: window config

AnchorContextMenu
Description: Name of the context menu that appears when the user right clicks on a hyperlink
Syntax: AnchorContextMenu menu-name
Default: AnchorContextMenu (IE default)
Context: window config

AutoComplete
Description: Determines whether AutoComplete is active
Syntax: AutoComplete On|Off
Default: AutoComplete Off
Context: window config

Bottom
Description: Distance of the window from the bottom of the offset rectangle
Syntax: Bottom pixels [px]|pixels sx|perctange%|points pt
Default: Bottom (undefined)
Context: window config

ControlContextMenu
Description: Name of the context menu that appears when the user right clicks on a form input field
Syntax: ControlContextMenu menu-name
Default: ControlContextMenu (IE default)
Context: window config

DefaultContextMenu
Description: Name of the context menu that appears when the user right clicks on an element that is neither a hyperlink, image, table, or input field
Syntax: DefaultContextMenu menu-name
Default: DefaultContextMenu (IE default)
Context: window config

DropDownMenu
Description: Specifies a menu to be used as the window's main drop-down menu
Syntax: DropDownMenu menu-name
Default: DropDownMenu (none)
Context: window config

ClientHeight
Description: Height of the window excluding the title bar, menu, and border–i.e. the height of the document view
Syntax: ClientHeight pixels [px]|pixels sx|perctange%|points pt
Default: ClientHeight (undefined)
Context: window config

ClientWidth
Description: Height of the window excluding its border–i.e. the width of the document view
Syntax: ClientWidth pixels [px]|pixels sx|perctange%|points pt
Default: ClientWidth (undefined)
Context: window config

<DialogBox>
Description: Contains directive affecting the appearance of a web browser window and indicates that should behave like a dialog box
Syntax: <DialogBox name-regex>
Context: server config

FlatScrollBar

Syntax: FlatScrollBar On|Off
Default: FlatScrollBar Off
Context: window config

Frameless
Description: Determines whether the window should be frameless

Default: Frameless Off
Context: window config

FullScreen

Default: FullScreen Off

Default: Height (undefined)

Syntax: HelpButton On|Off
Default: HelpButton Off
Context: window config

Default: MaximizeButton On StandardWindow

Default: Right (undefined)

Default: RoundedCorners 0 0

Syntax: DOMEnvMenuItem label window-name env-name env-value element-id element-property property-value

Syntax: DOMMenuItem label window-name element element-property property-value

Syntax: <menu name> ... <menu>

Syntax: MenuSeparator
Context: menu config

ScriptMenuItem

UrlMenuItem

AutoExit
Description: Determines whether the program exits automatically when all windows have closed
Syntax: AutoExit On|Off
Default: AutoExit Off
Context: system tray icon config

DispatchCharset
Description: Character-set used for translating menu command arguments to Unicode
Syntax: DispatchCharset charset
Default: DispatchCharset iso-8859-1
Context: system tray icon config

DispatchDomain
Description: Domain name used for expanding environment variables in menu command arguments
Syntax: DispatchDomain domain-name
Default: DispatchDomain (ServerName)
Context: system tray icon config

DoubleClickMenu

RightClickMenu

<SystemTrayIcon>
Description: Contains directives defining a System Tray icon

Syntax: ToolTip text
